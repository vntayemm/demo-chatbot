# FEATURES LISTING - Enterprise Chatbot | AI

## 1. Tong quan tinh nang

`demo-chatbot` la giai phap chatbot production-ready, gom 2 nhom tinh nang:
- Tinh nang san pham (chatbot)
- Tinh nang van hanh/governance (de go-live an toan)

## 2. Tinh nang chatbot cot loi

1. **Price bot**
   - Hoi dap thong tin goi gia, so sanh goi, uu dai.
2. **Guide bot**
   - Huong dan cai dat, huong dan su dung, thao tac co ban.
3. **Semantic retrieval**
   - Tra loi dua tren ngu canh tai lieu, khong chi theo keyword.
4. **top_k control**
   - Dieu chinh so luong ngu canh truy xuat trong moi query.
5. **API-first**
   - Endpoint tach rieng theo domain (`/chat/price`, `/chat/guide`).

## 3. Tinh nang giao dien

1. **Frontend test UI (Vue)**
   - Nhap cau hoi va xem cau tra loi theo tung bot.
2. **Landing pages**
   - Trang gioi thieu giai phap (`landing.html`, `landing_bilingual.html`).
3. **Lead capture form**
   - Thu thap thong tin khach hang tren landing song ngu.
4. **Blogs listing page**
   - Trang tong hop bai viet marketing (`blogs.html`).

## 4. Tinh nang van hanh va governance

1. **UAT checklist**
   - Nghiem thu theo tieu chi pass/fail ro rang.
2. **Go-live checklist**
   - Kiem soat dieu kien GO/NO-GO truoc production.
3. **SLA/SLO framework**
   - Do luong uptime, latency, error rate, quality benchmark.
4. **Risk register**
   - Quan tri rui ro theo muc do + owner + mitigation.
5. **Incident runbook**
   - Xu ly su co P1/P2/P3 + RCA.
6. **Sign-off templates**
   - UAT sign-off, Go-live sign-off, closure sign-off.
7. **Maintenance policy**
   - Chinh sach bao tri va SLA phan hoi sau ban giao.

## 5. Tinh nang thuong mai va pre-sales

1. **Bao gia da cap**
   - Day du, rut gon, song ngu, USD reference.
2. **Mail chao gia**
   - Mau tong hop + theo goi A/B/C + song ngu.
3. **Hop dong mau**
   - Ban day du, rut gon, song ngu VI-EN.
4. **Kick-off playbook**
   - Agenda, RACI, minutes, action follow-up.
5. **Pitch pack**
   - Elevator pitch, talk track 2p/5p, slide-ready.

## 6. Tinh nang tai lieu ky thuat

1. **System Design Document (SDD)**
2. **Architecture diagrams**
   - Landscape, HLA, Sequence, Workflow, ERD, HA, Scale, Network, Security, User Role.
3. **Requirement analysis + BPMN**
4. **Project plan + Gantt**
   - Tong the va theo goi A/B/C.
