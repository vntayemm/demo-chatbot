Sau khi đã đóng gói (log) pipeline chatbot vào MLflow, bạn triển khai (serve) chatbot như sau:

Xác định đường dẫn model đã log, ví dụ: runs:/<run_id>/chatbot_vietnamese.
Dùng lệnh sau để khởi động server phục vụ chatbot:
bash


mlflow models serve -m runs:/<run_id>/chatbot_vietnamese --port 5000

Gửi truy vấn tới endpoint /invocations bằng HTTP POST với dữ liệu JSON, ví dụ:
bash


curl -X POST http://127.0.0.1:5000/invocations \
  -H "Content-Type: application/json" \
  -d '("question": "Sản phẩm này giá bao nhiêu?", "context": "Nội dung markdown về sản phẩm...")'

Chatbot sẽ trả về kết quả dựa trên pipeline tiếng Việt bạn đã đóng gói.
