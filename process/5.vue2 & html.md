Bạn có thể tích hợp chatbot MLflow vào website sử dụng Vue.js 2 + HTML bằng cách gọi API MLflow từ frontend. Dưới đây là hướng dẫn chi tiết:

Đảm bảo MLflow serve đang chạy, ví dụ: http://localhost:5000/invocations.
Trong component Vue.js, tạo hàm gửi câu hỏi đến API:
javascript


methods: (
  async askChatbot() {
    const response = await fetch("http://localhost:5000/invocations", {
      method: "POST",
      headers: { "Content-Type": "application/json" ),
      body: JSON.stringify({
        question: this.userQuestion,
        context: this.contextData
      }),
    });
    const result = await response.json();
    this.botAnswer = result.answer || JSON.stringify(result);
  }
}

Trong template HTML:
html


<div>
  <input v-model="userQuestion" placeholder="Nhập câu hỏi..." />
  <button @click="askChatbot">Gửi</button>
  <div v-if="botAnswer">Trả lời: {{ botAnswer }}</div>
</div>

Đảm bảo xử lý CORS trên server MLflow nếu frontend và backend khác domain.



Dưới đây là ví dụ đầy đủ về cách tích hợp chatbot MLflow vào website sử dụng Vue.js 2 + HTML:

1. Đảm bảo MLflow serve đang chạy:

bash


mlflow models serve -m runs:/<run_id>/chatbot_vietnamese --port 5000

2. Tạo component Vue.js 2:

html


<!-- index.html -->
<div id="app">
  <input v-model="userQuestion" placeholder="Nhập câu hỏi..." />
  <button @click="askChatbot">Gửi</button>
  <div v-if="botAnswer">Trả lời: ({ botAnswer )}</div>
</div>

<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
<script>
new Vue({
  el: '#app',
  data: {
    userQuestion: '',
    contextData: 'Nội dung markdown về sản phẩm...', // hoặc lấy từ backend/website
    botAnswer: ''
  },
  methods: {
    async askChatbot() {
      try {
        const response = await fetch("http://localhost:5000/invocations", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({
            question: this.userQuestion,
            context: this.contextData
          }),
        });
        const result = await response.json();
        // Tuỳ vào cấu trúc trả về của model, bạn có thể cần điều chỉnh dòng dưới
        this.botAnswer = result.answer || JSON.stringify(result);
      } catch (error) {
        this.botAnswer = "Lỗi kết nối chatbot!";
      }
    }
  }
});
</script>

3. Lưu ý về CORS:
Nếu website và MLflow serve chạy trên hai domain/port khác nhau, bạn cần cấu hình CORS cho MLflow serve.

Cách đơn giản nhất là dùng proxy hoặc cấu hình CORS trên server backend (Flask/FastAPI).
Tham khảo: MLflow CORS Issue & Workaround
4. Kết quả:
Người dùng nhập câu hỏi, nhấn Gửi, chatbot sẽ trả lời ngay trên giao diện website.
