Semantic search là kỹ thuật tìm kiếm dựa trên ý nghĩa (ngữ nghĩa) thay vì chỉ dựa vào từ khóa. Trong MLflow, bạn có thể xây dựng chatbot hoặc hệ thống hỏi đáp semantic search bằng cách sử dụng Sentence Transformers để chuyển đổi câu hỏi và các đoạn văn bản (từ markdown, tài liệu...) thành vector embedding, sau đó tính toán độ tương đồng (cosine similarity) giữa embedding của câu hỏi và các đoạn văn để tìm ra nội dung liên quan nhất.

MLflow hỗ trợ log, quản lý và triển khai các mô hình Sentence Transformers, đồng thời cho phép bạn xây dựng custom pipeline semantic search, ví dụ: encode dữ liệu, lưu embedding, khi có câu hỏi thì encode câu hỏi và tìm kiếm đoạn văn bản gần nhất. Bạn muốn hướng dẫn chi tiết từng bước hoặc ví dụ mã nguồn không?


Dưới đây là hướng dẫn chi tiết xây dựng semantic search với Sentence Transformers và MLflow cho chatbot tiếng Việt (hoặc bất kỳ ngôn ngữ nào):

1. Cài đặt thư viện cần thiết
bash


pip install mlflow sentence-transformers pandas

2. Chuẩn bị dữ liệu
Giả sử bạn có file markdown hoặc danh sách các đoạn văn bản (ví dụ: các mục FAQ, blog, hướng dẫn).

3. Tạo embedding cho dữ liệu và log model với MLflow
python


import mlflow
from sentence_transformers import SentenceTransformer, util
import pandas as pd

# Bước 1: Load mô hình Sentence Transformers (chọn model hỗ trợ tiếng Việt nếu có)
model = SentenceTransformer("VoVanPhuc/sup-SimCSE-VietNamese-phobert-base")

# Bước 2: Chuẩn bị dữ liệu
documents = [
    "MLflow là nền tảng mã nguồn mở để quản lý vòng đời machine learning.",
    "Để cài đặt MLflow, bạn sử dụng pip install mlflow.",
    "Hướng dẫn sử dụng sản phẩm được ghi trong file hướng_dan.md.",
    # ... thêm các đoạn văn bản khác
]

# Bước 3: Encode dữ liệu thành embedding
corpus_embeddings = model.encode(documents, convert_to_tensor=True)

# Bước 4: Log model và dữ liệu với MLflow
with mlflow.start_run():
    mlflow.sentence_transformers.log_model(
        model=model,
        artifact_path="semantic_search_model",
        input_example=["Ví dụ câu hỏi tiếng Việt"],
    )
    # Lưu corpus để dùng cho truy vấn sau này
    pd.DataFrame(("documents": documents)).to_csv("corpus.csv", index=False)
    mlflow.log_artifact("corpus.csv")

4. Xây dựng hàm semantic search
python


from sentence_transformers import util
import torch

def semantic_search(query, model, corpus, corpus_embeddings, top_k=3):
    query_embedding = model.encode(query, convert_to_tensor=True)
    hits = util.semantic_search(query_embedding, corpus_embeddings, top_k=top_k)[0]
    results = [(corpus[hit['corpus_id']], hit['score']) for hit in hits]
    return results

# Ví dụ sử dụng:
query = "Cách cài đặt MLflow như thế nào?"
results = semantic_search(query, model, documents, corpus_embeddings)
for doc, score in results:
    print(f"Đoạn liên quan: {doc} (score: {score:.2f})")
