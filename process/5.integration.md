Để tích hợp chatbot MLflow vào website hoặc CRM, bạn cần triển khai model thành REST API (bằng mlflow models serve) và kết nối frontend (website/CRM) với API này. Website/CRM sẽ gửi câu hỏi của người dùng qua HTTP POST đến endpoint /invocations, nhận lại câu trả lời và hiển thị lên giao diện.

Các bước cơ bản:

Triển khai chatbot bằng MLflow serve (ví dụ trên port 5000).
Ở frontend (React, Vue, hoặc CRM platform), tạo hàm gửi request:
javascript


// Ví dụ với JavaScript/React
async function askChatbot(question, context) (
  const response = await fetch("http://localhost:5000/invocations", {
    method: "POST",
    headers: { "Content-Type": "application/json" ),
    body: JSON.stringify({ question, context }),
  });
  const result = await response.json();
  return result;
}

Khi người dùng nhập câu hỏi, gọi hàm này và hiển thị kết quả lên website/CRM.
Bạn cần đảm bảo bảo mật (CORS, xác thực) nếu triển khai thực tế. Nếu muốn tích hợp sâu vào CRM (ví dụ Salesforce, Odoo...), bạn cũng dùng HTTP POST tương tự hoặc xây dựng connector tùy nền tảng.
