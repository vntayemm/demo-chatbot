Dưới đây là hướng dẫn chi tiết từng bước để xây dựng hai chatbot riêng biệt cho CRM bằng MLflow và mô hình LLM hỗ trợ tiếng Việt:

Chuẩn bị dữ liệu:

Chatbot so sánh giá: Thu thập markdown từ landing page, blog, website về sản phẩm, giá cả.
Chatbot hướng dẫn sử dụng: Thu thập markdown hướng dẫn cài đặt, sử dụng sản phẩm.
Chọn mô hình LLM tiếng Việt:

Tìm mô hình hỗ trợ tiếng Việt trên Hugging Face (ví dụ: PhoBERT, vi-LLM, v.v.).
Tải về và kiểm thử khả năng trả lời tiếng Việt.
Đóng gói và log mô hình với MLflow:

Định nghĩa pipeline hoặc custom PythonModel/PyFunc cho từng chatbot, chỉ định nguồn dữ liệu phù hợp.
Sử dụng mlflow.pyfunc.log_model() để log từng chatbot riêng biệt, mỗi chatbot là một model artifact khác nhau.
Triển khai chatbot:

Dùng mlflow models serve để phục vụ từng chatbot trên các endpoint khác nhau.
Tích hợp endpoint vào CRM hoặc website, định tuyến truy vấn đến đúng chatbot theo mục đích.
Kiểm thử và đánh giá:

Tạo bộ câu hỏi kiểm thử cho từng chatbot.
Sử dụng MLflow Tracing/Evaluation để theo dõi chất lượng trả lời, điều chỉnh dữ liệu hoặc mô hình khi cần.
Ví dụ mã nguồn (giả lập):

python


import mlflow
from transformers import pipeline

# Chatbot so sánh giá
price_bot = pipeline("question-answering", model="mymodel-vietnamese")
mlflow.pyfunc.log_model("price_bot", python_model=price_bot)

# Chatbot hướng dẫn sử dụng
guide_bot = pipeline("question-answering", model="mymodel-vietnamese")
mlflow.pyfunc.log_model("guide_bot", python_model=guide_bot)

Sau đó, bạn triển khai từng chatbot:
#   bash
mlflow models serve -m runs:/<run_id>/price_bot --port 5001
mlflow models serve -m runs:/<run_id>/guide_bot --port 5002
