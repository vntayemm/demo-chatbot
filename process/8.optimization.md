Có vẻ để tối ưu semantic search cho dữ liệu lớn, bạn nên:

Tiền xử lý và lưu sẵn embedding của toàn bộ corpus vào file hoặc database (không encode lại mỗi lần truy vấn).
Sử dụng các vector database hoặc thư viện tối ưu như FAISS, Annoy, Milvus để tìm kiếm gần đúng (approximate nearest neighbor) thay vì so sánh tuần tự toàn bộ embedding.
Khi truy vấn, chỉ encode câu hỏi, sau đó dùng FAISS/Annoy để tìm top-k embedding gần nhất, rồi mới lấy nội dung tương ứng.
MLflow hỗ trợ log và quản lý model, còn việc tối ưu truy vấn lớn sẽ cần tích hợp thêm FAISS hoặc vector DB ngoài


#   FAISS
Để kết hợp MLflow với FAISS cho semantic search dữ liệu lớn, bạn thực hiện các bước sau:

Tạo embedding và lưu FAISS index

Encode toàn bộ corpus bằng Sentence Transformers.
Dùng FAISS để xây dựng index và lưu lại index này ra file.
Log model với MLflow

Log mô hình Sentence Transformers với MLflow.
Lưu file FAISS index và file chứa corpus (danh sách văn bản) dưới dạng artifact.
Triển khai custom pipeline

Khi phục vụ, load lại model, FAISS index và corpus.
Khi nhận câu hỏi, encode câu hỏi thành embedding, dùng FAISS để tìm top-k văn bản gần nhất.
Ví dụ mã nguồn:

python


import mlflow
from sentence_transformers import SentenceTransformer
import faiss
import numpy as np
import pandas as pd

# 1. Encode corpus và xây dựng FAISS index
model = SentenceTransformer("VoVanPhuc/sup-SimCSE-VietNamese-phobert-base")
documents = [...]  # danh sách văn bản
corpus_embeddings = model.encode(documents, convert_to_numpy=True)
index = faiss.IndexFlatL2(corpus_embeddings.shape[1])
index.add(corpus_embeddings)
faiss.write_index(index, "faiss.index")
pd.DataFrame(("documents": documents)).to_csv("corpus.csv", index=False)

# 2. Log model và artifact với MLflow
with mlflow.start_run():
    mlflow.sentence_transformers.log_model(
        model=model,
        artifact_path="semantic_search_model"
    )
    mlflow.log_artifact("faiss.index")
    mlflow.log_artifact("corpus.csv")

Triển khai custom pipeline (PythonModel):

python


import mlflow.pyfunc
import faiss
import pandas as pd
from sentence_transformers import SentenceTransformer

class FaissSemanticSearch(mlflow.pyfunc.PythonModel):
    def load_context(self, context):
        self.model = SentenceTransformer("VoVanPhuc/sup-SimCSE-VietNamese-phobert-base")
        self.index = faiss.read_index(context.artifacts["faiss_index"])
        self.corpus = pd.read_csv(context.artifacts["corpus_file"])["documents"].tolist()

    def predict(self, context, model_input):
        query = model_input["question"][0]
        query_emb = self.model.encode([query], convert_to_numpy=True)
        D, I = self.index.search(query_emb, k=3)
        return [self.corpus[i] for i in I[0]]

# Log custom pipeline
mlflow.pyfunc.log_model(
    python_model=FaissSemanticSearch(),
    artifact_path="faiss_semantic_search",
    artifacts={
        "faiss_index": "faiss.index",
        "corpus_file": "corpus.csv"
    }
)

#   API
có thể triển khai REST API cho pipeline semantic search sử dụng MLflow serve như sau:

Sau khi đã log custom pipeline (ví dụ: artifact_path="faiss_semantic_search"), bạn chạy:
bash


mlflow models serve -m runs:/<run_id>/faiss_semantic_search --port 5000

Gửi truy vấn từ website/CRM hoặc Postman:
bash


curl -X POST http://localhost:5000/invocations \
  -H "Content-Type: application/json" \
  -d '("question": ["Tôi muốn biết cách cài đặt sản phẩm"])'

Kết quả trả về là danh sách các đoạn văn bản liên quan nhất từ corpus.
